Longhand
const welcome = 'You have logged in as ' + first + '' + last + ''
const db = 'http://' + host + ':' + port + '/' + database

Shorthand
const welcome = `You have logged in as ${first} ${last}` 
const db = `http://${host}:${port}/${database}`

Other Shorthands are 
    Arrow Functions
    Implicit return Shorthand
    Object Property Shorthand
    Decimal Base etc...
